# Numerical and Analytical Methods for Engineers

This repository contains the development environment for the course **Numerical and Analytical Methods for Engineers**. It is designed to run seamlessly using GitHub Codespaces, providing a consistent Python-based computational environment for all students.

## Prerequisites

To use this environment, you need:
1. A GitHub account (free)
2. A web browser (Chrome, Firefox, Safari, or Edge recommended)
3. Internet connection

## Getting Started

### Step 1: Create a GitHub Account

If you don't already have a GitHub account:

1. Go to [https://github.com/signup](https://github.com/signup)
2. Enter your email address and click **Continue**
3. Create a password and username
4. Complete the verification process
5. Verify your email address

### Step 2: Launch GitHub Codespace

Once you have access to the repository:

1. Click the green **Code** button
2. Select the **Codespaces** tab
3. Click **Create codespace on main** (or the current branch)
4. Wait for the environment to build (first time takes a few minutes)
5. The VS Code editor will open in your browser with everything ready to use
